# 😂 Joke Bot

A simple Python-based Joke Bot that fetches and displays random jokes from an external API directly in the terminal.  
This project demonstrates API integration, environment variable usage, and basic error handling in Python.

---

## Features

- Fetches random jokes instantly
- Supports multiple joke categories
- Lightweight and easy to run
- Uses environment variables for security
- Beginner-friendly Python project
- Fast terminal-based interaction

---

## Joke Categories

The bot can generate different types of jokes, such as:

- General jokes  
- Programming jokes  
- Dark humor  
- Dad jokes / puns  
- Random jokes  
- Edgy humor (may contain mature themes)

**Note:** Some jokes may include mature or sensitive humor. User discretion is advised.

---

## Technologies Used

- Python
- Requests library
- REST API
- Environment variables (.env)
